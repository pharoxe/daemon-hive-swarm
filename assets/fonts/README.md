# Fonts

`ProtoMono-SemiBold.ttf` is bundled for headings and mirrored into
`android/app/src/main/assets/fonts/` so React Native can resolve it by family name.

Body/UI text uses Jura variants from `E:\Fonts\jura`: `Jura-Medium.ttf` for body
copy and `Jura-DemiBold.ttf` for buttons and compact controls.
